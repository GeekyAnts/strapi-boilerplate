## To start with Example Project

### Clone the Project

```
git clone https://github.com/code-ayush/strapi-example.git
```

or

```
git clone git@github.com:code-ayush/strapi-example.git
```

### Install dependency using

```
yarn || npm install
```

### Start the project

```
yarn develop || npm run develop
```
